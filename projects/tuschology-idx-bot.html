Tuschology-IDX-Bot

Overview

Tuschology-IDX-Bot is an automated stock news aggregator bot that collects and delivers the latest Indonesian financial market headlines directly to a Telegram channel. Built to help retail investors stay informed without having to manually check multiple news sources throughout the day.

Problem

Indonesian retail investors often miss important market-moving news because financial information is scattered across multiple sites and portals. Manually tracking these sources every few hours is time-consuming and easy to miss.

Solution

The bot runs on a scheduled cron job, scraping news from trusted Indonesian financial news sources. It filters relevant stock-related articles, deduplicates them, and pushes formatted summaries to a Telegram channel with ticker symbols highlighted.

Key Features

  • Scrapes multiple Indonesian financial news sources every 30 minutes
  • Deduplication engine to avoid sending the same news twice
  • Stock ticker detection and tagging in headlines
  • Formatted Telegram messages with article links
  • Simple SQLite database to track sent articles
  • Error handling and retry logic for failed requests

Technical Challenges

The main challenge was handling anti-scraping measures on some news sites and dealing with inconsistent HTML structures across sources. I implemented rotating request headers and a fallback RSS parser for sites that support it.

Another challenge was building a reliable deduplication system — the same news often gets republished with slightly different titles across portals. I ended up using a fuzzy string matching approach based on headline similarity.

What I Learned

This project taught me a lot about web scraping reliability, scheduling in Python, and working with the Telegram Bot API. It's been running for over 6 months with minimal downtime.
