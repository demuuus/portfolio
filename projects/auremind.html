AureMind

Overview

Auremind is an AI-powered note-taking web application designed to help students study more effectively through structured knowledge management and intelligent assistance. Inspired by Notion and Obsidian, the platform combines traditional note organization with AI-driven features such as concept definition, semantic search, and automated note generation.

Project Brief

The project focuses on building a modern web-based study tool that enhances how students capture, organize, and retrieve knowledge. It integrates AI capabilities into the note-taking workflow, allowing users to not only store information but also interact with it dynamically. The system emphasizes usability, scalability, and efficient information retrieval.

What Was Built

A full-stack web application that allows users to create, edit, and organize notes in a structured environment. The platform incorporates AI features to assist users in generating notes, defining concepts, and quickly searching across their knowledge base. It supports a flexible note structure similar to block-based editors and interconnected knowledge systems.

Key Features

  • AI-powered note generation and summarization
  • Context-aware definitions for study concepts
  • User authentication and profile management
  • Organize tasks and personal schedules
  • Clean and responsive web interface
  • Semantic search across all notes
  • Create, edit, and delete notes
  • Django-powered backend

Database Design

The schema was designed to 3rd Normal Form to eliminate redundancy. Key tables include Students, Courses, Enrollments, Users, and AuditLog. Foreign key constraints and transactions ensure data integrity across operations.

Outcome

Received full marks for the project. The main feedback from the lecturer was positive on the normalization and transaction handling, with a suggestion to add indexing on the frequently queried columns — which I implemented as an extra improvement.
